Welcome to the timestamp microservice!

This application takes as a parameter either a unix timestamp or a natural date and returns a JSON object containing both the unix timestamp representation and the natural representation of that date. Anything not in either of those formats will return null values for the object properties.

Example Usage:

https://murmuring-journey-10632.herokuapp.com/December%2015,%202015

https://murmuring-journey-10632.herokuapp.com/1450137600


Example Output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
